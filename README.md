# Progetto_Esame

L’applicazione sviluppata è una piattaforma di domotica completa, composta da un sistema integrato frontend e backend, progettata per il controllo intelligente di dispositivi domestici come luci, tapparelle, sensori e prese smart. Il sistema consente la gestione dei dispositivi tramite un’interfaccia intuitiva e una comunicazione in tempo reale, garantita dal backend. L’implementazione dei comandi vocali e l’integrazione con Amazon Alexa sono state realizzate in collaborazione con 4Usolution.
